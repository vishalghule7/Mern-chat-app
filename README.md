## Setup

###  setup .env File

Create a `.env` file in the root of your project and add the following variables:

```makefile
PORT=...
MONGO_DB_URI=...
JWT_SECRET=...
NODE_ENV=...
```
Build the app
```
    npm run build
```
Start the app
```
    npm start
```
